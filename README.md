# 永劫无间战绩查询插件

## 简介

这是一个基于 **naraka.drivod.top API** 的永劫无间战绩查询插件，支持查询玩家的赛季统计数据和最近对战记录。

## 功能特点

- ✅ 支持多种游戏模式筛选（天选、天人系列）
- ✅ 显示赛季统计数据（段位、分数、对局数、胜率等）
- ✅ 显示最近 10 场对战详细记录
- ✅ 显示每场对战的详细数据（击杀、伤害、治疗、振刀、武器、魂玉等）
- ✅ 显示队友数据（队友英雄、击杀、伤害、治疗）
- ✅ 无需配置 cookie，开箱即用
- ✅ 美化的输出界面，使用 emoji 增强视觉效果

## 使用方法

### 命令格式

```
/yj [模式] <角色名>
```

### 可用模式

| 模式 | 模式代码 | 说明 |
|------|----------|------|
| `天选单排` | 1 | 天选之人单排 |
| `天选双排` | 12 | 天选之人双排 |
| `天选三排` | 2 | 天选之人三排 |
| `天人单排` | 4 | 天人之战单排 |
| `天人双排` | 13 | 天人之战双排 |
| `天人三排` | 5 | 天人之战三排 |

### 示例

```
/yj 冬眠岛              # 查询全部模式战绩
/yj 天选三排 冬眠岛      # 查询天选三排战绩
/yj 天人双排 冬眠岛      # 查询天人双排战绩
```

## 输出说明

### 标题信息
- **角色名**：玩家在游戏中的角色名称
- **模式**：当前查询的游戏模式

### 赛季统计
- **段位**：当前段位名称和分数
- **屠龙**：赛季屠龙次数
- **对局数**：赛季总对局数
- **第一数**：赛季获得第一名的次数
- **前五数**：赛季进入前五的次数
- **第一率**：第一名胜率
- **前五率**：前五胜率
- **场均击败**、**场均助攻**、**场均治疗**
- **场伤**、**K/D**、**最多振刀**
- **最高击败**、**最高伤害**、**最高治疗**

### 对战信息
- **时间**：对局结束时间（MM-DD HH:MM）
- **地图**：对局地图名称
- **英雄**：使用的英雄名称
- **模式**：对局模式
- **排名**：最终排名
- **击杀**：击杀数
- **伤害**：总伤害值
- **评级**：对局评级（S/A/B/C）
- **分差**：段位分变化

### 详细数据
- **生存时间**、**治疗量**
- **振刀次数**、**招式命中数**
- **释放技能次数**、**爆头数**
- **救援/复活次数**、**开堆数**
- **移动距离**、**消耗暗潮币**
- **武器**：使用的武器及伤害统计
- **魂玉**：装备的魂玉及等级
- **护甲**：护甲等级
- **称号**：获得的称号

### 队友数据
- **英雄名(角色名)**：队友使用的英雄和角色名
- **击杀**：队友击杀数
- **伤害**：队友伤害值
- **治疗**：队友治疗量

- 妖刀姬、崔三娘、岳山、无尘、顾清寒、武田信忠
- 殷紫萍、沈妙、胡为、季莹莹、玉玲珑、哈迪
- 刘炼、张起灵、蓝梦、万钧、李寻欢、巫真、席拉、哈迪·伊斯马尔

## 注意事项

1. **无需配置**：插件无需任何 cookie 配置，开箱即用
2. **API 限制**：请合理使用，避免频繁查询
3. **数据延迟**：战绩数据可能存在一定的延迟
4. **角色搜索**：使用角色名进行搜索，请确保角色名正确
